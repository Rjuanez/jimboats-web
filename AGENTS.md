# AGENTS.md

Este archivo existe para que Codex cargue automaticamente las normas de este
proyecto web.

Antes de hacer cualquier trabajo en este repositorio, leer completo:

- `AGENT.md`

`AGENT.md` es la fuente de verdad de las normas del proyecto y exige leer
completos:

- `architecture.md`
- `implementation.md`
- `stack.md`
- `testing.md`
- `ui-ux.md`
- `workflow.md`

Estas normas tambien aplican cuando el directorio de trabajo sea la carpeta
superior `JimBoats`, porque el producto activo del workspace es esta web.

Si falta alguno de esos archivos, o si alguna instruccion entra en conflicto,
parar y pedir aclaracion antes de cambiar archivos.
